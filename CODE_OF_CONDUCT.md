# Code of Conduct

## Deutsch

Wir möchten eine freundliche, sachliche und inklusive Projektumgebung. Bitte kommunizieren Sie respektvoll, bleiben Sie konstruktiv und vermeiden Sie persönliche Angriffe. Problematisches Verhalten kann über die GitHub-Kanäle des Projekts gemeldet werden.

## English

We want a friendly, focused, and inclusive project environment. Please communicate respectfully, stay constructive, and avoid personal attacks. Problematic behavior can be reported through the project's GitHub channels.
