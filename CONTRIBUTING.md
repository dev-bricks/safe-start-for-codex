# Contributing

## Deutsch

Beiträge sind willkommen. Bitte öffnen Sie zuerst ein Issue, wenn eine Änderung das Verhalten beim Pausieren, Freigeben oder Wiederherstellen von Automationen betrifft.

Richtlinien:

- Keine hardcoded lokalen Pfade.
- Keine Binaries, Shortcuts oder privaten Snapshots committen.
- Änderungen an Dateisystem- oder Prozesslogik mit Tests absichern.
- Sicherheitsrelevante Probleme nicht öffentlich melden; siehe `SECURITY.md`.

## English

Contributions are welcome. Please open an issue first if a change affects how automations are paused, released, or restored.

Guidelines:

- Do not add hardcoded local paths.
- Do not commit binaries, shortcuts, or private snapshots.
- Cover filesystem and process-handling changes with tests.
- Do not report security issues publicly; see `SECURITY.md`.
